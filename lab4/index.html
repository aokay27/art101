Lab 4 - Pseudocoding

Challenge

This lab's challenge was to create some pseudocode based on an everyday task and a simple computer game.

Problems

We did not really run into any serious problems during this lab, as the instructions were very clear and easy to follow along with.

Reflection

I think this assignment went fairly well and helped us learn the importance of pseudocoding and how it can make breaking a complicated task down into easier parts worthwhile.
